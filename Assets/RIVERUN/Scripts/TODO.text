# CUANDO EL RÍO SUENA

## Concepto del juego

**Cuando el río suena** es un endless runner en tercera persona donde el jugador debe escapar de una enorme crecida de agua que lo persigue constantemente.

El jugador controla a un explorador/minero que, al intentar encontrar un diamante escondido bajo tierra, provoca accidentalmente la salida de un río subterráneo. Ahora debe correr por un sendero junto al río mientras evita obstáculos, recoge piedras y escapa antes de que el agua lo alcance.

---

# Historia

Un antiguo minero busca un diamante perdido cerca de un río.

Después de varios golpes con su pico consigue romper la roca donde supuestamente estaba escondida la gema. Pero en lugar de encontrar un tesoro, rompe una fuente de agua subterránea.

El agua comienza a salir con fuerza y rápidamente se convierte en una enorme riada.

El minero debe escapar por el sendero mientras el río crece detrás de él.

**"Cuando el río suena..."**

# Cámara

El juego utiliza una cámara en tercera persona.

La cámara está colocada detrás del personaje:

```
        Cámara

          📷


          🏃
          ↓
       dirección
       del camino


        🌊
```

El jugador corre alejándose de la cámara mientras el escenario aparece delante.

---

# Mecánicas principales

## Movimiento

El personaje corre automáticamente.

El jugador puede:

* Moverse al carril izquierdo.
* Mantenerse en el centro.
* Moverse al carril derecho.
* Saltar obstáculos.

Controles:

```
A / Flecha izquierda
Mover izquierda

D / Flecha derecha
Mover derecha

ESPACIO
Saltar
```

---

# Sistema de carriles

El camino tiene 3 posiciones:

```
Izquierda       Centro       Derecha


   🪵             🏃            🪵


  X=-8            X=0          X=8
```

El jugador puede cambiar entre estos tres carriles.

---

# Obstáculos

Los obstáculos principales son troncos.

Ejemplo:

```
🌲      🌲

       🏃


🪵          🪵
```

Cuando el jugador golpea un tronco:

* Pierde velocidad durante 1 segundo.
* Se tropieza.
* El río se acerca.

Después vuelve a correr normalmente.

---

# Coleccionables

Durante el camino aparecen piedras/gemas.

El jugador puede recogerlas para conseguir puntos.

Ejemplo:

```
       💎

🏃
```

Cuando toca una piedra:

* La piedra desaparece.
* Aumenta la puntuación.

---

# Sistema de dificultad

El juego aumenta la dificultad progresivamente.

Cambios:

* El río acelera.
* Aparecen más obstáculos.
* Hay menos tiempo para reaccionar.
* Los patrones de piedras son más difíciles.

---

# Río perseguidor

El río es la mecánica principal.

El agua aparece detrás del jugador y avanza constantemente.

Ejemplo:

```
Inicio:

🏃



🌊


Más tarde:


🏃


🌊🌊🌊


Final:


🌊🌊🌊🌊
🏃
```

Si el río alcanza al jugador:

```
GAME OVER
```

# Intro cinematográfica

Antes de comenzar la partida aparece una pequeña escena.

Duración aproximada:

10-15 segundos.

## Escena 1

El minero encuentra una zona misteriosa.

Ve algo brillante bajo una roca.

```
      💎

      🪨

      ⛏️
```

---

## Escena 2

Sonidos:

* Golpes del pico.
* Piedras cayendo.

La roca se rompe.

---

## Escena 3

Pero en realidad:

```
💎 ❌

💧
💧💧
💧💧💧
```

El personaje mira atrás.
